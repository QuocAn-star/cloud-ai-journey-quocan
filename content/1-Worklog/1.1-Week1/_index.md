**Duration:** 20/04/2026 - 24/04/2026

## Week 1 Objectives

- Become familiar with the internship environment and the Amazon Web Services (AWS) ecosystem.
- Learn the fundamental AWS services used for developing and deploying cloud-based applications.
- Set up an AWS account, configure the working environment, and practice using essential AWS services.
- Build a solid foundation in cloud computing to prepare for the development of the project in the following weeks.

### Tasks to Be Completed This Week

| Day | Task | Date | Reference |
|:---:|------|:----:|-----------|
| **Monday** | Study the fundamentals of Amazon Web Services (AWS), explore the concepts of Cloud Computing, and learn about the major AWS service categories, including Compute, Storage, Networking, Database, and Security, to build a solid foundation for the internship. | 20/04/2026 | https://aws.amazon.com/what-is-aws/ |
| **Tuesday** | Create an AWS Free Tier account, become familiar with the AWS Management Console, explore the service navigation structure, and practice using the basic features required for future development tasks. | 21/04/2026 | https://docs.aws.amazon.com/awsconsolehelpdocs/latest/gsg/what-is.html |
| **Wednesday** | Learn about Amazon EC2, including virtual server concepts, instance types, Amazon Machine Images (AMIs), Key Pairs, and Security Groups, and launch the first EC2 instance for hands-on practice. | 22/04/2026 | https://docs.aws.amazon.com/AWSEC2/latest/UserGuide/concepts.html |
| **Thursday** | Explore Amazon Bedrock by learning about Generative AI services, Foundation Models, and how AI capabilities can be integrated into cloud-based applications on AWS. | 23/04/2026 | https://docs.aws.amazon.com/bedrock/latest/userguide/what-is-bedrock.html |
| **Friday** | Study AWS Budgets to understand cloud cost monitoring and management, and learn the fundamentals of AWS Lambda and the Serverless Computing model for developing applications without managing servers. | 24/04/2026 | https://docs.aws.amazon.com/cost-management/latest/userguide/budgets-managing-costs.html<br>https://docs.aws.amazon.com/lambda/latest/dg/welcome.html |

## Weekly Achievements

- Gained a solid understanding of Cloud Computing concepts and the AWS ecosystem.
- Successfully created an AWS Free Tier account and configured the AWS Management Console environment.
- Completed hands-on exercises with Amazon EC2, Amazon Bedrock, AWS Budgets, AWS Lambda, and Amazon RDS.
