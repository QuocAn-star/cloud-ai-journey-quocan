**Duration:** 04/05/2026 - 08/05/2026

## Week 3 Objectives

- Understand the fundamental networking concepts on the AWS Cloud platform.
- Learn the role of Amazon VPC in building secure and scalable cloud network infrastructures.
- Practice creating and configuring a Virtual Private Cloud (VPC) and its related components.
- Gain hands-on experience in controlling network traffic and securing AWS resources.

## Weekly Achievements

- Gained a solid understanding of AWS networking concepts and the architecture of Amazon VPC.
- Successfully created and configured a Virtual Private Cloud (VPC), including Public and Private Subnets.
