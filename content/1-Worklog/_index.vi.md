Phần này tổng hợp quá trình thực tập của mình trong 12 tuần với định hướng chính là xây dựng **FinOps-Optimized Serverless Medallion Data Lakehouse Architecture for Customer Behavior Analytics trên nền tảng AWS**.

Chương trình thực tập kéo dài **12 tuần** (20/04/2026 - 10/07/2026), bao gồm các nội dung từ kiến thức nền tảng về AWS đến thiết kế, xây dựng, triển khai và hoàn thiện một hệ thống phân tích dữ liệu theo kiến trúc **Medallion Data Lakehouse**.

---

| Tuần | Khoảng thời gian | Nội dung tập trung |
|------|-----------------|-------------------|
| [**Tuần 1**](1.1-Week1/) | 20/04 - 24/04 | Làm quen chương trình thực tập, tìm hiểu AWS Cloud, AWS Management Console, Amazon EC2, Amazon Bedrock, AWS Budgets và AWS Lambda |
| [**Tuần 2**](1.2-Week2/) | 27/04 - 01/05 | Nghiên cứu Amazon S3, Storage Classes, Versioning, Lifecycle và AWS IAM phục vụ lưu trữ và quản lý dữ liệu |
| [**Tuần 3**](1.3-Week3/) | 04/05 - 08/05 | Tìm hiểu Amazon VPC, Subnet, Route Table, Internet Gateway, Security Group và Network ACL |
| [**Tuần 4**](1.4-Week4/) | 11/05 - 15/05 | Phân tích bài toán Customer Behavior Analytics, nghiên cứu Medallion Architecture và thiết kế kiến trúc Data Lakehouse trên AWS |
| [**Tuần 5**](1.5-Week5/) | 18/05 - 22/05 | Xây dựng Data Ingestion Pipeline, triển khai Batch Processing, Streaming Processing, Amazon Kinesis Data Firehose và Bronze Layer |
| [**Tuần 6**](1.6-Week6/) | 25/05 - 29/05 | Xây dựng AWS Glue ETL, làm sạch dữ liệu, chuẩn hóa dữ liệu và triển khai Silver Layer |
| [**Tuần 7**](1.7-Week7/) | 01/06 - 05/06 | Xây dựng Gold Layer, tổng hợp dữ liệu phân tích và cấu hình AWS Glue Data Catalog |
| [**Tuần 8**](1.8-Week8/) | 08/06 - 12/06 | Thiết lập Amazon Athena, xây dựng các truy vấn SQL và chuẩn bị dữ liệu phục vụ Dashboard |
| [**Tuần 9**](1.9-Week9/) | 15/06 - 19/06 | Phát triển Dashboard bằng Streamlit, kết nối Amazon Athena và trực quan hóa dữ liệu |
| [**Tuần 10**](1.10-Week10/) | 22/06 - 26/06 | Triển khai Dashboard trên Amazon EC2, cấu hình môi trường và kiểm thử khả năng truy cập hệ thống |
| [**Tuần 11**](1.11-Week11/) | 29/06 - 03/07 | Tích hợp toàn bộ Data Lakehouse Pipeline, kiểm thử End-to-End và tối ưu hiệu năng hệ thống |
| [**Tuần 12**](1.12-Week12/) | 06/07 - 10/07 | Hoàn thiện hệ thống, viết báo cáo thực tập, chuẩn bị Demo và tổng kết dự án |
