During my internship at **AWS** with a focus on **Cloud Engineering / Data Engineering**, I concentrated on developing my personal project: **FinOps-Optimized Serverless Medallion Data Lakehouse Architecture for Customer Behavior Analytics on AWS**.

The project allowed me to gain practical experience in building a complete Data Lakehouse solution, from data ingestion through Batch Processing and Streaming Processing, data transformation using AWS Glue ETL, organizing data with the Medallion Architecture, querying data through Amazon Athena, and visualizing business insights using a Streamlit dashboard deployed on Amazon EC2.

The main tasks I completed include:

- Designed a Serverless Medallion Data Lakehouse architecture on AWS.
- Built two independent data ingestion pipelines:
  - Batch Processing.
  - Streaming Processing.
- Used Amazon S3 as the Data Lake with Raw, Bronze, Silver, and Gold layers.
- Developed AWS Glue ETL Jobs to transform data across the Medallion layers.
- Configured AWS Glue Data Catalog to manage metadata.
- Queried analytical datasets using Amazon Athena.
- Developed an interactive business analytics dashboard using Streamlit.
- Deployed the dashboard on Amazon EC2.
- Prepared the project documentation using the Hugo Workshop Website, including Worklogs, Proposal, Workshops, Blogs, Events, Self Evaluation, and Feedback.
- Added supporting screenshots from AWS Console, including Amazon S3, AWS Glue, Amazon Athena, Amazon EC2, and the deployed dashboard.
