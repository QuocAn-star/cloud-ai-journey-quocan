This section summarizes the 12-week internship journey, focusing on the development of a **FinOps-Optimized Serverless Medallion Data Lakehouse Architecture for Customer Behavior Analytics on AWS**.

The internship lasted **12 weeks** (20/04/2026 - 10/07/2026), covering topics from AWS fundamentals to designing, building, deploying, and completing a data analytics system based on the **Medallion Data Lakehouse Architecture**.

---

| Week | Duration | Focus Area |
|------|----------|------------|
| [**Week 1**](1.1-Week1/) | 20/04 - 24/04 | Introduction to the internship program, AWS Cloud fundamentals, AWS Management Console, Amazon EC2, Amazon Bedrock, AWS Budgets, and AWS Lambda |
| [**Week 2**](1.2-Week2/) | 27/04 - 01/05 | Studied Amazon S3, Storage Classes, Versioning, Lifecycle policies, and AWS Identity and Access Management (IAM) for secure data storage |
| [**Week 3**](1.3-Week3/) | 04/05 - 08/05 | Learned Amazon VPC, Subnets, Route Tables, Internet Gateway, Security Groups, and Network ACLs |
| [**Week 4**](1.4-Week4/) | 11/05 - 15/05 | Analyzed the Customer Behavior Analytics use case, studied the Medallion Architecture, and designed the AWS Data Lakehouse architecture |
| [**Week 5**](1.5-Week5/) | 18/05 - 22/05 | Built the data ingestion pipeline, implemented batch and streaming processing, configured Amazon Kinesis Data Firehose, and developed the Bronze Layer |
| [**Week 6**](1.6-Week6/) | 25/05 - 29/05 | Developed AWS Glue ETL jobs, cleaned and transformed data, standardized datasets, and implemented the Silver Layer |
| [**Week 7**](1.7-Week7/) | 01/06 - 05/06 | Built the Gold Layer, generated analytical datasets, and configured the AWS Glue Data Catalog |
| [**Week 8**](1.8-Week8/) | 08/06 - 12/06 | Configured Amazon Athena, developed SQL queries, and prepared datasets for the analytics dashboard |
| [**Week 9**](1.9-Week9/) | 15/06 - 19/06 | Developed the Streamlit dashboard, integrated Amazon Athena, and created data visualizations |
| [**Week 10**](1.10-Week10/) | 22/06 - 26/06 | Deployed the dashboard on Amazon EC2, configured the runtime environment, and validated system accessibility |
| [**Week 11**](1.11-Week11/) | 29/06 - 03/07 | Integrated the complete Data Lakehouse pipeline, performed end-to-end testing, and optimized overall system performance |
| [**Week 12**](1.12-Week12/) | 06/07 - 10/07 | Finalized the system, completed the internship report, prepared the project demonstration, and summarized the internship outcomes |
